# Abilities

## Shield (Passive)
- [ ] Blocks 50% damage from the first hit every round

## Chain Lightning
- [ ] Deals base + WIS
- [ ] Base hit * WIS
- [ ] Multihits changes target then back to original target

## Fire Missiles
- [ ] Deals base + WIS
- [ ] Base hit * AGI

## Fireball
- [ ] Deals base + 2*WIS
